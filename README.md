## ğŸ® Usage

The portfolio features an interactive terminal interface. Type commands to navigate:

- `help` - Show list of available commands
- `about` - Display profile summary and bio
- `projects` - List all projects
- `project <name>` - Show details for a specific project (e.g., `project kmrl`)
- `skills` - Display technical skills dashboard
- `experience` - View work & leadership experience timeline
- `ctfs` - Show CTF achievements and competitions
- `certifications` - List professional certifications
- `contact` - Open contact information modal
- `clear` - Reset to landing view and clear terminal

**Tips:**
- Use Tab for autocomplete
- Commands are case-sensitive
- Type `help` anytime to see available commands
